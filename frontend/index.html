❓ Quick Start Guide

🚀 Getting Started

Welcome! Here's the recommended order of operations:

1. Add Suppliers

Click 🏭 Add Supplier — suppliers must exist before you can add products.

2. Add Distributors

Distributors are needed before placing orders. Add them via the FAB (+) button.

3. Add Products

Link each product to a supplier.

4. Place Orders

Select a product and a distributor.

5. Create Shipments

Link a shipment to an existing order and set the delivery status.

💡 Tips

  • Click any editable cell and hit 💾 to save inline changes
  • Use Graphs for visual analytics
  • Use ↻ Refresh to sync latest DB data
